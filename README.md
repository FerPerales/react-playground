# React playground

An exercise to play with React done in 4 hours aprox.

## Objective

Working on components abstraction mainly and how those components
interact.

## Setup

```
bower install
npm install
```

## Start development serve


```
  mkdir dist
  grunt serve
```

Site will be available in `localhost:9000`
